Nome do Curso

Atualizações nos novos templates de cursos de 2021

De maneira experimental foram adicionadas novas classes pra auxiliar os diagramadores na tarefa de diagramar, neste arquivo está uma compilação das novidades. Apenas os cursos de 2021 contam com estes recursos, mas eles sao portáveis os templates antigos linkando os arquivos corretamente. As classes antigas seguem valendo caso não desejem utilizar as novas aqui definidas, mas certamente essas novas classes facilitam a criação de novos icones que nao constam no template, por exemplo. A sintaxe para usar icones é mais simples, e contamos com classes adicionais para usar as cores da paleta definidas para a UC.

As novas classes e definições encontram-se no arquivo extras.css(aonde é feito um import com os arquivos icones_default.css icones_apoio.css, modal-100.css e custom-accordion.css)
